title: "Embedded chat widget"
---

On the [Publish pane](https://docs.langflow.org/concepts-publish), the **Embed into site** tab displays code that can be inserted in the `<body>` of your HTML to interact with your flow.

The chat widget is implemented as a web component called `langflow-chat` and is loaded from a CDN. For more information, see the [langflow-embedded-chat repository](https://github.com/langflow-ai/langflow-embedded-chat).

For a sandbox example, see the [Langflow embedded chat CodeSandbox](https://codesandbox.io/p/sandbox/langflow-embedded-chat-example-dv9zpx).

The following example includes the minimum required inputs, called [props](https://react.dev/learn/passing-props-to-a-component) in React, for using the chat widget in your HTML code, which are `host_url` and `flow_id`. The `host_url` value must be `HTTPS`, and may not include a `/` after the URL. The `flow_id` value is found in your Langflow URL. For a Langflow server running the [Basic prompting flow](https://docs.langflow.org/basic-prompting) at `https://c822-73-64-93-151.ngrok-free.app/flow/dcbed533-859f-4b99-b1f5-16fce884f28f`, your chat widget code is:

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@langflow/embedded-chat@latest/dist/index.js"></script>
<langflow-chat
  host_url="https://c822-73-64-93-151.ngrok-free.app"
  flow_id="dcbed533-859f-4b99-b1f5-16fce884f28f">
</langflow-chat>
```

When this code is embedded within HTML, it becomes a responsive chatbot, powered by the basic prompting flow.

## Chat widget with style customization

To configure your chat widget further, include additional props.

All props and their types are listed in [index.tsx](https://github.com/langflow-ai/langflow-embedded-chat/blob/main/src/index.tsx).

```html
<langflow-chat
  host_url="https://c822-73-64-93-151.ngrok-free.app"
  flow_id="dcbed533-859f-4b99-b1f5-16fce884f28f"
  chat_window_style='{"border-radius":"10px","box-shadow":"0 0 10px rgba(0,0,0,0.1)"}'
  bot_message_style='{"background-color":"#f0f0f0","color":"#333","border-radius":"10px"}'
  user_message_style='{"background-color":"#4CAF50","color":"white","border-radius":"10px"}'>
</langflow-chat>
```

To add a custom [session ID](https://docs.langflow.org/session-id) and an API key:

```html
<langflow-chat
  host_url="https://c822-73-64-93-151.ngrok-free.app"
  flow_id="dcbed533-859f-4b99-b1f5-16fce884f28f"
  session_id="custom-session-123"
  api_key="your-api-key-here">
</langflow-chat>
```

> Note: Your flow must contain **Chat Input** and **Chat Output** components.

## Embed in React

To use the chat widget in your React application, create a wrapper component like this:

```tsx
import React, { useEffect, useRef } from 'react';

interface ChatWidgetProps {
  className?: string;
  flowId?: string;
  hostUrl?: string;
  apiKey?: string;
  sessionId?: string;
}

export const ChatWidget: React.FC<ChatWidgetProps> = ({
  className,
  flowId = "your-default-flow-id",
  hostUrl = "https://your-langflow-server.com",
  apiKey,
  sessionId,
  ...props
}) => {
  const scriptLoaded = useRef(false);

  useEffect(() => {
    if (!scriptLoaded.current) {
      const script = document.createElement('script');
      script.src = 'https://cdn.jsdelivr.net/npm/@langflow/embedded-chat@latest/dist/index.js';
      script.type = 'module';
      document.body.appendChild(script);
      scriptLoaded.current = true;
    }
  }, []);

  return (
    <langflow-chat
      className={className}
      flow_id={flowId}
      host_url={hostUrl}
      api_key={apiKey}
      session_id={sessionId}
      {...props}
    />
  );
};
```

Example usage:

```tsx
import { ChatWidget } from './components/ChatWidget';

function App() {
  return (
    <div className="App">
      <ChatWidget
        flowId="dcbed533-859f-4b99-b1f5-16fce884f28f"
        hostUrl="https://c822-73-64-93-151.ngrok-free.app"
        className="my-chat-widget"
      />
    </div>
  );
}
```

## Embed in Angular

Add `CUSTOM_ELEMENTS_SCHEMA` to your Angular module:

```ts
import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule],
  providers: [],
  bootstrap: [AppComponent],
  schemas: [CUSTOM_ELEMENTS_SCHEMA]
})
export class AppModule {}
```

Use the component:

```ts
import { Component, OnInit } from '@angular/core';

@Component({
  selector: 'app-chat-widget',
  template: `
    <langflow-chat
      host_url="https://c822-73-64-93-151.ngrok-free.app"
      flow_id="dcbed533-859f-4b99-b1f5-16fce884f28f"
      [chat_window_style]="chatWindowStyle"
      [bot_message_style]="botMessageStyle">
    </langflow-chat>
  `
})
export class ChatWidgetComponent implements OnInit {
  chatWindowStyle = {
    'border-radius': '10px',
    'box-shadow': '0 0 10px rgba(0,0,0,0.1)'
  };

  botMessageStyle = {
    'background-color': '#f0f0f0',
    'color': '#333',
    'border-radius': '10px'
  };

  ngOnInit(): void {
    const script = document.createElement('script');
    script.type = 'module';
    script.src = 'https://cdn.jsdelivr.net/npm/@langflow/embedded-chat@latest/dist/index.js';
    document.body.appendChild(script);
  }
}
```

## Chat widget configuration

Props with the type `JSON` need to be passed as stringified JSON (i.e., `'{"key":"value"}'`).

| Prop                    | Type    | Description                                                    |
| ----------------------- | ------- | -------------------------------------------------------------- |
| `flow_id`               | String  | Required. The flowâ€™s unique ID.                                |
| `host_url`              | String  | Required. Base URL of Langflow server (HTTPS, no trailing `/`) |
| `api_key`               | String  | Optional API key for auth.                                     |
| `session_id`            | String  | Custom session ID.                                             |
| `additional_headers`    | JSON    | Extra HTTP headers.                                            |
| `height`, `width`       | Number  | Dimensions of the widget.                                      |
| `chat_position`         | String  | `top-left`, `bottom-right`, etc.                               |
| `start_open`            | Boolean | Whether the chat window opens automatically.                   |
| `chat_window_style`     | JSON    | Full widget styling.                                           |
| `chat_trigger_style`    | JSON    | Trigger button styling.                                        |
| `bot_message_style`     | JSON    | Bot message appearance.                                        |
| `user_message_style`    | JSON    | User message appearance.                                       |
| `error_message_style`   | JSON    | Error styling.                                                 |
| `input_style`           | JSON    | Chat input box.                                                |
| `input_container_style` | JSON    | Chat input container.                                          |
| `send_button_style`     | JSON    | Send button.                                                   |
| `send_icon_style`       | JSON    | Icon for send button.                                          |
| `window_title`          | String  | Widget header title.                                           |
| `placeholder`           | String  | Input placeholder.                                             |
| `placeholder_sending`   | String  | Placeholder while sending.                                     |
| `online`                | Boolean | Toggle online state.                                           |
| `online_message`        | String  | Message shown when online.                                     |
| `input_type`            | String  | Chat input type.                                               |
| `output_type`           | String  | Chat output type.                                              |
| `output_component`      | String  | Which component to route output to.                            |
| `chat_output_key`       | String  | Output key to render.                                          |
| `tweaks`                | JSON    | Custom tweaks for flow.                                        |
